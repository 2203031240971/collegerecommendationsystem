# 🎓 EduMatch — AI-Powered College Recommendation Platform

An intelligent system designed to help students discover the best-fit colleges based on **marks, budget, preferences, and interests**, solving the real-world problem of confusing and time-consuming college selection. 

---

## 🚀 Overview

EduMatch simplifies the college selection process by combining:

* 📊 Smart scoring algorithms
* 🤖 AI-powered guidance
* 🌐 Multi-state college data

It enables students to make **data-driven and personalized decisions** instead of relying on guesswork.

---

## ✨ Key Highlights

* 🎯 **50+ Colleges Across India** (Hyderabad, Bangalore, Delhi, Mumbai, Pune, Chennai & more)
* ⚡ **Dual Modes**

  * Local Database (Fast & ready-to-use)
  * Real-time Search (Live data using APIs)
* 🤖 **AI Chat Advisor** for guidance on colleges, fees & placements
* 📊 **Smart Recommendation Engine** using a 4-factor scoring model
* 📱 **Responsive UI** (works on desktop & mobile)

---

## 🧠 How It Works

The system evaluates colleges using a weighted scoring algorithm:

* 📈 Marks Compatibility — 30%
* ❤️ Interest Match — 30%
* 📍 Location Preference — 20%
* 💰 Budget Fit — 20%

### 🎯 Recommendation Categories

* 🟢 **Safe** — High probability of admission
* 🟡 **Match** — Moderate probability
* 🔴 **Reach** — Competitive options

## 🚀 Getting Started

### 1. Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2. Run Frontend

```bash
python -m http.server 3000 --directory frontend
```

### 3. Open in Browser

```
http://localhost:3000/index.html
```
